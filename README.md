## Overview

以 Python 實作的 Telegram Bot 德州撲克遊戲，支援玩家在 Telegram 聊天室中與具有三種不同策略的電腦 AI 對戰，涵蓋完整四輪下注流程（Pre-flop、Flop、Turn、River）與自動結算機制。

## Features

| Feature | Description |
|---|---|
| 三種 AI 策略 | 保守 / 平衡 / 激進，策略差異明顯 |
| 完整四輪下注 | Pre-flop → Flop → Turn → River |
| 自動結算 | 牌型判斷與籌碼自動計算 |

## Tech stack

- **Language：** Python
- **Platform：** Telegram Bot API（python-telegram-bot）
